Code blocks are rendered with syntax highlighting out of the box.

## Basic Code Block

```python
def fibonacci(n: int) -> int:
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)
```

## With Title

```ts title="utils.ts"
export function slugify(text: string): string {
  return text
    .toLowerCase()
    .replace(/\s+/g, "-")
    .replace(/[^\w-]+/g, "");
}
```

## With Line Highlighting

```js title="config.js" {3-5}
const config = {
  name: "My Docs",
  // These lines are highlighted
  theme: "dark",
  lang: "en",
  output: "static",
};
```

## Inline Code

Use `backticks` for inline code like `const x = 1` or `pip install`.
