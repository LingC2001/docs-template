---
title: File Tree
description: Display directory structures with files and folders.
---

The file tree component visually represents directory structures. Useful for showing project layouts or explaining where files should go.

## Basic File Tree

<Files>
  <Folder name="src" defaultOpen>
    <Folder name="components" defaultOpen>
      <File name="Button.tsx" />
      <File name="Header.tsx" />
      <File name="Footer.tsx" />
    </Folder>
    <Folder name="lib">
      <File name="utils.ts" />
      <File name="api.ts" />
    </Folder>
    <File name="index.ts" />
  </Folder>
  <File name="package.json" />
  <File name="tsconfig.json" />
  <File name="README.md" />
</Files>

## Project Structure Example

<Files>
  <Folder name="docs" defaultOpen>
    <Folder name="app" defaultOpen>
      <Folder name="(home)">
        <File name="layout.tsx" />
        <File name="page.tsx" />
      </Folder>
      <Folder name="docs">
        <File name="layout.tsx" />
      </Folder>
      <File name="layout.tsx" />
      <File name="global.css" />
    </Folder>
    <Folder name="content" defaultOpen>
      <Folder name="docs" defaultOpen>
        <File name="index.mdx" />
        <File name="getting-started.mdx" />
        <File name="meta.json" />
      </Folder>
    </Folder>
    <File name="package.json" />
    <File name="source.config.ts" />
  </Folder>
</Files>

## Usage

```mdx
<Files>
  <Folder name="src" defaultOpen>
    <File name="index.ts" />
    <Folder name="utils">
      <File name="helpers.ts" />
    </Folder>
  </Folder>
  <File name="package.json" />
</Files>
```

### Props

| Component  | Prop          | Type      | Description                     |
| ---------- | ------------- | --------- | ------------------------------- |
| `Folder`   | `name`        | `string`  | Folder name                     |
| `Folder`   | `defaultOpen` | `boolean` | Start expanded                  |
| `File`     | `name`        | `string`  | File name                       |
