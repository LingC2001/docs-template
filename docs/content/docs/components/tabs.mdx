Tabs let users switch between different views of related content.

## Usage

```mdx
<Tabs items={["Tab 1", "Tab 2"]}>
  <Tab value="Tab 1">
    Content for the first tab.
  </Tab>
  <Tab value="Tab 2">
    Content for the second tab.
  </Tab>
</Tabs>
```

### Props

| Prop            | Type       | Description                             |
| --------------- | ---------- | --------------------------------------- |
| `items`         | `string[]` | Tab labels                              |
| `defaultIndex`  | `number`   | Initially selected tab (default: `0`)   |
