Markdown tables work out of the box with styling from Fumadocs.

## Basic Table

| Name       | Type     | Description              |
| ---------- | -------- | ------------------------ |
| `title`    | `string` | Page title               |
| `author`   | `string` | Author name              |
| `date`     | `string` | Publication date         |
| `draft`    | `bool`   | Whether this is a draft  |
